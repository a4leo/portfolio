2
Prototyping and wireframes

The first step was planning the interaction between the dashboard with the app. We need it to think like
our client and see what were the changes that they would want to do inside the dashboard and how this
changes would be seeing inside the app.

We made this part the most important in our meetings with our clients. With the help of everybody involved,
we were able to create a good start for the project.

3
Visual design

With the wireframes finally approved, we were able to explore and have fun with the most creativity part of
the project, giving our best to create and appealing look but still respecting the guidelines of the company.

4
Mobile interface

The app was meant for a different public, it was no longer only for our client -the crew of the hotel- but also
for the customers of the hotel. It need to have an easy and friendly use for the user so we decide that it need
to have the flow of what most people use know days, a social media app.

5
Style and design

The style guide is directly connected with the guidelines given by the marketing department of the hotel.
It focus on the same color, typography and visual direction.
